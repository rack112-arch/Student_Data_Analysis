# Student Performance Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on student performance data using Python.

The objective is to analyze score patterns, identify correlations, and generate insights through data visualization.

---

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

---

## Key Features
1 Data Cleaning  
2 Column Standardization  
3 Statistical Analysis  
4 Data Visualization  
5 Correlation Heatmap  
6 Insight Generation  

---

## Key Insights
- Students scored higher in reading and writing compared to math.
- Strong positive correlation exists between reading and writing scores.
- Performance trends suggest that students good in one subject tend to perform well in others.
